Web App Development

Front-End Applications using HTML, Java & CSS

Project Overview

Designed and developed multiple interactive web applications and static sites using HTML, Java, and CSS. Projects included a fully responsive restaurant website, an interactive trivia game, and a rule-based chatbot for user assistance. Focused on delivering clean, user-friendly interfaces and demonstrating front-end logic integration.

Key Responsibilities

  • Developed responsive front-end interfaces using HTML5, CSS3, and JavaScript (DOM manipulation and API calls).
  • Built a restaurant demo website showcasing dynamic menu rendering and order simulation logic.
  • Created a rule-based chatbot to simulate user support and guided responses.
  • Ensured accessibility and cross-browser compatibility across multiple devices.
  • Used GitHub Pages for deployment and version control.

Technologies Used

  • HTML5, CSS3, and JavaScript (ES6)
  • OpenAI API (for chatbot enhancement)
  • GitHub Pages for hosting and versioning
  • VS Code and Chrome DevTools for debugging and optimization

Impact & Results

  • Developed functional, user-friendly web apps from scratch using only OpenAI guidance.
  • Improved front-end skills and learned responsive design principles hands-on.
  • Demonstrated the effectiveness of AI-assisted learning and project development.
