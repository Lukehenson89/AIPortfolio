Digital Transformation Analyst

Secro Middle East – Supply Chain & ERP Implementation

Overview

Led a digital transformation initiative focused on transitioning supply chain operations from SAP to Oracle Fusion Cloud. Managed API integrations, ETL development, and UAT support for critical ERP modules impacting forecasting and procurement.

Key Achievements

  • Successfully migrated 2TB of operational data across multiple business systems.
  • Built custom ETL pipelines and REST API connections for real-time data flow.
  • Coordinated cross-departmental UAT and managed incident resolutions via Jira.
  • Reduced data latency by 85% and improved accuracy of planning forecasts.

Technologies Used

  • Oracle Fusion Cloud, SQL Server, Power BI
  • Oracle Integration Cloud (OIC), Postman, Jira, Confluence
