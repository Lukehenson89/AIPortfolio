ETL & API Integration Project

System Migration from SAP to Oracle | Supply Chain & Demand Planning

Project Overview

AS Digital transformation was picked up by the BI team, this project involved the end-to-end migration of supply chain and demand planning data pipelines from SAP BW to Oracle Fusion Cloud. The focus was on building robust ETL workflows, integrating APIs for real-time data synchronization, and ensuring data accuracy across multiple systems during and after migration.

Key Responsibilities

  • Designed and implemented ETL processes to extract master and transactional data from SAP, transform for Oracle data models, and load into staging environments.
  • Developed API integrations for real-time exchange of order, inventory, and forecast data between Oracle Fusion, Power BI, and third-party logistics systems.
  • Collaborated with Oracle technical teams to troubleshoot integration errors and optimize API endpoints for performance.
  • Managed mapping documentation, data lineage tracking, and transformation logic across multiple functional areas.
  • Supported UAT for data reconciliation, ensuring data flows aligned with new business rules and planning hierarchies.
  • Provide post go live support nd maintenance alongside rollout training.

Technologies & Tools Used

  • SQL Server, and Oracle PL/SQL for ETL scripting and validation
  • Oracle Integration Cloud (OIC) and REST APIs for system interoperability
  • Power BI and Excel for data visualization and reconciliation reporting
  • Postman for API testing and automation
  • Jira and Confluence for sprint tracking, documentation, and UAT sign-off
  • Microsoft Teams and SharePoint for collaboration and status reporting

Impact & Results

  • Successfully migrated over 2TB of supply chain and planning data with zero data loss.
  • Enabled real-time inventory visibility through REST API integration between Oracle and warehouse systems.
  • Reduced manual data processing time by 90% via automated ETL pipelines.
  • Improved forecast accuracy through synchronized data between planning and ERP modules.
  • Delivered a stable data foundation that enhanced decision-making and supply chain agility post go-live.
